# Mission: 成为 AI Agent 开发工程师

## Why
高阳是一名前端工程师，目标是转型为 AI Agent 开发工程师。需要从零开始，系统学习 LLM 原理、Agent 架构、工具调用、本地知识库、项目落地等全链路技能。以 Hermes Agent（Nous Research 开源模型）为学习载体，贯穿整个学习过程。最终能独立设计、开发、部署 AI Agent 应用。

## Success looks like
- 能独立在本地部署并运行 Hermes 等开源模型，理解模型选型逻辑
- 能用 **TypeScript** 构建完整的 tool-calling Agent，集成多个外部 API
- 能搭建本地 RAG 知识库系统：文档解析 → 向量化 → 检索引擎 → LLM 回答
- 能设计 Agent 应用架构：记忆系统、多工具编排、流式交互（全 TS 类型安全）
- 能交付一个完整项目：本地知识库 + Agent 对话应用（可作为作品集）
- 能跟进 2026 年最新技术：MCP 协议、Vercel AI SDK、Agent Memory 等

## Constraints
- 角色：前端工程师（JS/TS 熟练，Python 不熟）
- 工具链：**TypeScript 优先**（Vercel AI SDK + LangChain.js TS + LanceDB TS + Ollama JS SDK）
- 核心模型：Hermes 3.1（通过 Ollama 本地运行）
- 学习节奏：每天 30-60 分钟，5 天/周，总周期 12 周
- 硬件：Windows 环境，优先轻量方案（量化模型）
- 教学语言：中文，技术术语保留英文

## Out of scope
- 模型训练/微调（学习重点是"用模型"，不是"训模型"）
- Python 生态（LangChain Python、PyTorch 等）
- Transformer 数学推导
- 纯学术论文研读
