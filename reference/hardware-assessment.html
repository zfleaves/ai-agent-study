结论

✅ 可以跑，但需要优化配置

能用，不是残废。16GB 内存 + i7 足够学习 AI Agent 开发。只是速度不会很快，但学习完全够用。

需要做的 3 个优化

1. 把 Ollama 模型存到 E 盘

C 盘只有 29GB 空闲，存模型太紧张。Ollama 默认存 C 盘，需要改到 E 盘：

# 1. 设置环境变量（PowerShell 管理员）
[Environment]::SetEnvironmentVariable('OLLAMA_MODELS', 'E:\ollama\models', 'User')

# 2. 重启 Ollama（任务栏右键 Ollama 图标 → Quit，再重新打开）

# 3. 验证
ollama pull hermes3:8b
# 模型会下载到 E:\ollama\models 而不是 C 盘

2. 用 Q4_K_M 量化版本（默认）

ollama pull hermes3:8b 默认下载的就是 Q4_K_M，质量/速度的最佳平衡点。不需要额外操作。

3. 跑模型时关闭其他应用

  • 关掉不需要的浏览器标签页（Chrome 吃内存大户）
  • VS Code 可以开着，但关掉不需要的终端
  • 模型推理时 CPU 会到 100%，这是正常的

  • 学习内容完全相同，只是回答质量稍差一点
