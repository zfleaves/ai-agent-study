二、覆盖率总评

当前覆盖率：~85%

核心链路（Function Calling + RAG + Agent 循环 + MCP）全部覆盖。三语言能力是显著优势。

三、需要补的 4 个缺口

# 缺口 为什么重要 建议
🔴 1 LangGraph #1 Agent 编排框架，JD 高频出现。有状态图、条件分支、人机协作 第 10 周增加 LangGraph 专题（TS 版 + Python 版）
🔴 2 Agent 评估体系 "如何评估 Agent 好不好"是面试高频题。LangSmith、Eval 指标 第 15 周增加 Agent Eval 专题
🟡 3 CrewAI + GraphRAG 多 Agent 框架 + 图 RAG 是 2026 年新趋势 第 10 周提 CrewAI，第 6 周提 GraphRAG 概念
🟡 4 Guardrails 安全护栏 生产 Agent 必须防止越权调用、幻觉输出 第 15 周工程化部分增加

五、调整建议

补上 4 个缺口后，覆盖率可达 ~95%
  1. 第 10 周增加 LangGraph 专题（1 天）：状态图、条件分支、人机协作
  2. 第 15 周增加 Agent Eval 专题（1 天）：评估指标、LangSmith、A/B 测试
  3. 第 6 周提 GraphRAG（30 分钟）：概念 + 适用场景
  4. 第 15 周增加 Guardrails（30 分钟）：工具调用权限、输出校验
