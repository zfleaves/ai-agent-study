# 目标调整：从前端到 AI Agent 全栈落地

用户在学完第 1 课后，明确了自己的真实目标：不只是学 Hermes Agent，而是以前端工程师身份切入 AI Agent 开发，最终能落地 LLM 项目（Agent 自动化 + 本地知识库 RAG）。

**Evidence**: 用户主动表述："我是前端，后续我想学习 ai agent 然后落地大模型或者其他的 LLM 等项目落地，和本地知识库搭建"。

**Implications**:
- 学习路线需要从"Hermes Agent 专项"扩展为"前端 → AI Agent 全栈"路径
- 工具链应优先选 JS/TS 生态（Vercel AI SDK、LangChain.js、Ollama），而非 Python
- 原有"Out of scope"中的"其他 Agent 框架"现在应纳入学习范围
- HERMES 学术框架的优先级进一步降低——实用性优先
- 课程需要重新编排：RAG 知识库、Agent 架构设计、前端集成等成为核心模块
- 第 1 课（Hermes 概览）仍然有价值，但定位调整为"Agent 生态的一个具体案例"
