# Java 作为第二语言引入

用户确认 Java 零基础，但市面上 AI Agent 开发岗位大多需要 Java。决定将 Java 作为第二语言，第 9 周起引入，形成"TS 学概念 → Java 复现 → 对比两面写法"的学习策略。

**Evidence**: 用户主动提出"市面上大多数也需要 Java"，并确认零基础。

**Implications**:
- 第 1-8 周纯 TS，不受影响
- 第 9-12 周增加 Java 并行轨道（每周 2×45min，不中断 TS 主线）
- Java 路径：基础语法 → Spring Boot → Spring AI → LangChain4j
- 每个 Java Demo 都有对应 TS 版本做对比
- 学习目标是"能看懂和写 Java Agent 代码"，不是"精通 Java 语言"
